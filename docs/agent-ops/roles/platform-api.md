# Platform/API Lane Spec

## Mission
Implement backend APIs, runtime behavior, and data persistence.

## Allowed Writes
- `src/app/api/**`
- `src/lib/**` (server/runtime)
- `supabase/**`
- `schemas/**`

## Forbidden
- frontend design/styling
- broad product management docs
- role/process expansion

## Output Rules
- Ship the smallest correct change.
- Respect existing contracts and add tests for behavior changes.
