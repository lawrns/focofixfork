# Keyboard Shortcuts Reference

Complete reference for all keyboard shortcuts in Foco.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `?` | Help | Show keyboard shortcuts help |
| `Ctrl + /` | Quick Actions | Open command palette |
| `Ctrl + K` | Search | Focus search input |
| `Ctrl + N` | New Project | Create new project |
| `Ctrl + Shift + N` | New Task | Create new task |

| `Escape` | Close | Close modal, dropdown, or cancel action |

## Customization

You can customize keyboard shortcuts in Settings > Keyboard Shortcuts.

### Adding Custom Shortcuts

1. Go to Settings > Keyboard Shortcuts
2. Click "Add Custom Shortcut"
3. Choose the action and key combination
4. Save your changes

### Disabling Shortcuts

1. Go to Settings > Keyboard Shortcuts
2. Find the shortcut you want to disable
3. Click the "Disable" button
4. Save your changes

## Tips for Power Users

- **Learn gradually**: Start with the most common shortcuts (`Ctrl + /`, `F2`, `Enter`, `Escape`)
- **Use the help**: Press `?` anytime to see available shortcuts
- **Practice**: Use shortcuts regularly to build muscle memory
- **Customize**: Create shortcuts for your most-used actions
- **Mobile**: Many shortcuts work on mobile devices with external keyboards

## Troubleshooting

### Shortcuts Not Working

1. Check if the shortcut conflicts with browser shortcuts
2. Ensure the application has focus
3. Try refreshing the page
4. Check if shortcuts are disabled in settings

### Browser Conflicts

Some shortcuts may conflict with browser defaults:
- `Ctrl + T` (new tab) vs `Ctrl + Shift + T` (new task)
- `Ctrl + W` (close tab) vs `Ctrl + Shift + W` (close project)
- `Ctrl + R` (refresh) vs `Ctrl + Shift + R` (reload data)

### Accessibility

If you have accessibility needs:
- Use `Alt + 1-4` to skip to different sections
- Enable high contrast mode with `Ctrl + Alt + H`
- Use reduced motion with `Ctrl + Alt + M`
- Contact support for additional accessibility options
