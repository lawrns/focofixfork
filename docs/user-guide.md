## Workflow
- Navigate to `/voice` and select the Voice tab.
- Enter instructions or use capture; press Generate Plan.
- Review tasks and milestones; assign owners and priorities.
- Switch to Timeline to validate sequencing; edit if needed.
- Commit plan to create a project with milestones and tasks.

## Collaboration
- Presence indicators show active teammates.
- Realtime updates for milestones and tasks via Supabase channels.
- Comments and mentions enhance coordination.

## Advanced Features
- Intelligent decomposition with validated outputs.
- Dependency hints derived from ordering and semantics.
- Versioning and diff/merge for plan iterations.

## Shortcuts
- Tab switch: Voice/Review/Timeline via button toggles.
- Keyboard-first editing in lists.

## Security
- RLS-enabled data access; SSL DB connections.
