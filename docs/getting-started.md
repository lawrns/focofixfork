# Getting Started with Foco

Welcome to Foco, the modern project management platform designed for teams that value productivity and collaboration. This guide will help you get up and running quickly.

## Quick Start

### 1. Create Your Account

1. Visit [foco.mx](https://foco.mx)
2. Click "Sign Up" and create your account
3. Verify your email address
4. Complete the onboarding tour

### 2. Set Up Your Organization

1. Create your first organization
2. Invite team members
3. Set up your workspace preferences
4. Configure notifications

### 3. Create Your First Project

1. Click "New Project" on the dashboard
2. Choose a template or start from scratch
3. Add team members to the project
4. Create your first tasks

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Open command palette |
| `Ctrl/Cmd + K` | Quick search |
| `Ctrl/Cmd + N` | New project |
| `Ctrl/Cmd + T` | New task |
| `Arrow Keys` | Navigate cards |
| `Enter` | Edit selected item |
| `Escape` | Cancel editing |

## Best Practices

### Project Organization
- Use consistent naming conventions
- Create clear project descriptions
- Set up proper labels and colors
- Regular project reviews

### Task Management
- Break down large tasks
- Use due dates effectively
- Assign clear ownership
- Regular status updates

### Team Collaboration
- Use @mentions for notifications
- Comment on tasks for context
- Share project updates regularly
- Use presence indicators
