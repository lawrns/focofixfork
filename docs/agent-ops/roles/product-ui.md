# Product/UI Lane Spec

## Mission
Implement user-facing UX and frontend behavior with clarity and polish.

## Allowed Writes
- `src/app/**` (non-API routes)
- `src/components/**`
- `public/**`
- frontend config/style files

## Forbidden
- DB migrations
- backend/API implementation
- role/process expansion

## Output Rules
- Ship code or ask one concrete clarifying question.
- Keep change scope narrow and reviewable in one pass.
